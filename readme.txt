Sidebar Widget to get carpooling results.

== Description ==

Sidebar Widget to get carpooling results mostly in germany and some european cities. 
start and/or destination can either be fixed in the backend or dynamically inserted by the user.

== Installation ==

1. Extract `mitfahrgelegenheiten-german.zip` to a local temporary directory
2. Upload the folder `mitfahrgelegenheiten-german` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag and drop widget into sidebar
5. If you like, enter a start and/or a destination. An autocompleter will show you possible locations
